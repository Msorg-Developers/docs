## Welcome

The Dancity merchant API (`/api/external/v1`) lets you buy airtime, data, cable TV subscriptions, electricity tokens, internet bundles, education PINs, and eSIMs — and fund betting accounts — all from your Dancity wallet balance. Every request is authenticated with your **merchant API key** and the `channel: API` header.

## Before you start

<Steps>
  <Step title="Dancity account (Tier 3)">
    Sign up at [dancity.app](https://dancity.app). API access requires **Tier 3** status and an approved API client.
  </Step>
  <Step title="Generate your API key">
    After approval, generate a merchant key from your [API Key page](https://dancity.app/profile/api-key). **Copy it once** — it is not shown again.
  </Step>
  <Step title="Allowlist your server IPs">
    Add the IP addresses of your backend servers to the allowlist. Requests from non-allowlisted IPs return `401`.
  </Step>
  <Step title="Set a webhook URL (optional)">
    Add an HTTPS webhook URL to receive real-time transaction status events. See the [Webhooks guide](/guides/webhook).
  </Step>
</Steps>

## Base URL

```
https://api.dancity.app
```

All endpoints are prefixed with `/api/external/v1/`.
