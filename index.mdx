## Welcome to Dancity API

The Dancity API lets approved partners integrate Nigeria's leading digital payment and value-added service (VAS) platform directly into their own products.

With a single API key you can:

## Before you start

<Steps>
  <Step title="Create a Dancity account">
    Sign up at [dancity.app](https://dancity.app) and upgrade to **Tier 3** — API access is only available to verified Tier 3 users.
  </Step>
  <Step title="Submit an API access request">
    From your profile page, navigate to **API Key** and submit an access request with your business name, website, and KYC documents.
  </Step>
  <Step title="Wait for approval">
    Our team reviews requests within 1–2 business days. You'll receive an email and in-app notification when approved.
  </Step>
  <Step title="Generate your API key">
    Once approved, generate your API key from the API Key page. **Copy it immediately** — it is shown only once.
  </Step>
  <Step title="Make your first request">
    Follow the [Quickstart](/guides/quickstart) guide to make your first API call in minutes.
  </Step>
</Steps>

## Base URL

All API requests are made to:

```
https://api.dancity.app
```
