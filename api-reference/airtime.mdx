## Buy Airtime

Purchase airtime for a single phone number.

```http
POST /airtime/buy
```

### Headers

| Header | Value | Required |
|--------|-------|----------|
| `Authorization` | `Bearer <api-key>` | âś… |
| `Content-Type` | `application/json` | âś… |
| `channel` | `API` | âś… |

### Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `product` | string | âś… | Network product name (e.g. `"MTN Airtime"`) |
| `phone` | string | âś… | 11-digit Nigerian phone number |
| `amount` | number | âś… | Amount in NGN. Min: `50`, Max: `10000` |
| `pin` | string | âś… | Your 4-digit transaction PIN |
| `customerRef` | string | âťŚ | Your own reference for idempotency tracking |
| `saveAsBeneficiary` | boolean | âťŚ | Save phone as a beneficiary |
| `beneficiaryName` | string | âťŚ | Label for the saved beneficiary |
| `couponCode` | string | âťŚ | Promotional coupon code |

### Example request

<CodeGroup>
```bash cURL
curl -X POST https://api.dancity.app/airtime/buy \
  -H "Authorization: Bearer dcy_live_xxxx..." \
  -H "Content-Type: application/json" \
  -H "channel: API" \
  -d '{
    "product": "MTN Airtime",
    "phone": "08012345678",
    "amount": 500,
    "pin": "1234",
    "customerRef": "ORDER-001"
  }'
```

## Bulk Airtime

Purchase airtime for multiple phone numbers in a single call. Counts as one request against your rate limit.

```http
POST /airtime/bulk
```

### Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `recipients` | array | âś… | Array of recipient objects (max 50) |
| `recipients[].phone` | string | âś… | 11-digit phone number |
| `recipients[].amount` | number | âś… | Amount in NGN per recipient. Min: `50` |
| `recipients[].product` | string | âś… | Network product name |
| `pin` | string | âś… | Your 4-digit transaction PIN |

### Example request

<CodeGroup>
```bash cURL
curl -X POST https://api.dancity.app/airtime/bulk \
  -H "Authorization: Bearer dcy_live_xxxx..." \
  -H "Content-Type: application/json" \
  -H "channel: API" \
  -d '{
    "pin": "1234",
    "recipients": [
      { "product": "MTN Airtime", "phone": "08012345678", "amount": 200 },
      { "product": "Airtel Airtime", "phone": "08087654321", "amount": 100 }
    ]
  }'
```

```javascript Node.js
const res = await fetch("https://api.dancity.app/airtime/bulk", {
  method: "POST",
  headers: {
    Authorization: `Bearer ${process.env.DANCITY_API_KEY}`,
    "Content-Type": "application/json",
    channel: "API",
  },
  body: JSON.stringify({
    pin: "1234",
    recipients: [
      { product: "MTN Airtime", phone: "08012345678", amount: 200 },
      { product: "Airtel Airtime", phone: "08087654321", amount: 100 },
    ],
  }),
});
```
</CodeGroup>
